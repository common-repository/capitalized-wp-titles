== Description ==

WP Capitalized Titles by http://www.easyguidetowp.com/

This is plugin will convert all your post and page titles into capitalized word format, Buzzfeed style.

== Installation ==

1. Upload `capitalized-wp-titles` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! Your titles should now be in uppercased word format
